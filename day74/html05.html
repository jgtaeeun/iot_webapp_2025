입력양식

항상 form 태그 안에 작성할 것.
input type에는 여러가지 많음(text, number 등)
버튼 클릭시 저장이나 동작이 발생함. input type="submit"이 일반적인 저장버튼임.

form의 데이터 전달방식은 get, post가 가장 많이 사용됨
get방식은 url로 데이터를 직접 입력해서 전달하는 방식
post방식은 별도의 방식(숨겨서)으로 데이터를 해당주소로 전달하는 방식
